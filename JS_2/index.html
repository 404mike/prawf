JavaScript - Working with third party library and events

Add four buttons to viewer.html (opens in new window). Each button should allow a user to click and rotate the image 90 degrees, 180 degrees, 360 degrees and 0 degrees.

To rotate the image call setRotation with JavaScript - eg viewer.viewport.setRotation(180)
