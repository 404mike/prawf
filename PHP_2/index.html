PHP 2 - Data parsing

people.json (opens in new window) contains information for ~500 people (randomly generated).

Each person has the following information: 

        {
          "id": 147,
          "name": "Dr. Garnet Kessler",
          "address": "645 Deron Radial Apt. 566\nPort Ricky, WI 93689",
          "email": "pblick@reinger.net",
          "profile": "images\/147.jpg"
        }

You must split the people.json (opens in new window) file into groups of 25 and place them within their own directory. Similar to: 

       - people
          - group_1
            - person_2
            - person_45
            ...
          - group_2
            - person_89
            - person_234
            ...
          - group_3
            ...
The number at the end of the person directory should be the users ID.

Each person directory should include a JSON file containing that persons information and their profile image. 

        - people
          - group_1
            - person_2
              - image_147.jpg
              - metadata.json
